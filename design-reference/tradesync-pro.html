TradeSync Pro

A real-time MT5 copy-trading platform that mirrors a master account to followers automatically — sizing every position correctly, on minimal infrastructure.

TYPECopy-trading platform
STATUSIn production

// CHALLENGE

The problem

Replicating trades across MetaTrader 5 accounts by hand is slow and error-prone. Every order has to be re-sized to each follower's balance and risk — and a single mistake in that math is a real loss, repeated on every trade.

It also doesn't scale. As the number of mirrored accounts grows, manual copying becomes impossible to do accurately or fast enough to matter in live markets.

// SOLUTION

What we built

A real-time system that mirrors trades from a master account to followers automatically, sizing each position to the follower's balance and risk settings. Lightweight Python desktop clients sit on each MT5 terminal and stream fills to a set of NestJS microservices; a Next.js dashboard gives operators live visibility — all designed to run on minimal infrastructure.

KEY DECISIONS
→

Desktop clients on each terminal to work within MT5's constraints, instead of fighting broker API limits.

→

Event-driven microservices, so one account's issue never cascades to the rest.

→

PostgreSQL as the single source of truth for every position and a full audit trail.

→

Stateless services on AWS to keep the platform cheap to run at near-zero idle cost.
